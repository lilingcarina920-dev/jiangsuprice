# 江苏每日价格采集系统

## 本地开发

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务
```bash
npm start
```

### 3. 访问
- 商户端：http://localhost:3000
- 管理后台：http://localhost:3000/admin.html

## 默认账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | admin123 |

## 功能

- 商户端H5：手机号验证 + 填写价格
- 管理后台：查看进度 + 导出数据
- 按月采集，支持本月补填
